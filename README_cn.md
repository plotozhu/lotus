

# Pnyx Consensus 普适共识 

这是“普适共识算法”的实现[论文]().

### 说明

分片是解决区块链的可扩展问题的最可行解决方案，但是要满足分片系统的共识算法需要具有六个必要条件：
1. 随机的区块倡议者，
2. 公平的经济激励系统
3. 低通信代价
4. 在异步通信网络下的活性保证
5. 快速最终确定性
6. 分片安全性。

目前大部分的共识算法无法兼顾上述六种需求，因此基于这些共识所实现的分片方案不具有实际的可行性。
